# Hardware Requirements

ravynOS should be able to run on most modern hardware from 2012.
Following are the hardware requirements to be able to run ravynOS:

 * UEFI boot
 * 4 GB of RAM
 * x86 CPU, 64-bit, 2 GHz dual-core Intel or AMD (4+ cores)<sup><b>*</b></sup>
 * Intel HD, UHD and Iris Xe Graphics lineups, or AMD Radeon graphics (GPUs currently compatible with FreeBSD and Wayland)<sup><b>**</b></sup>
 * PS2 & USB Devices (I2C devices may not work)
 * Wired or wireless network controllers compatible with FreeBSD-Current<sup><b>***</b></sup>

To check support for individual components, refer the [FreeBSD hardware support list](https://www.freebsd.org/releases/12.0R/hardware/)
