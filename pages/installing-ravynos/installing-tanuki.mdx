# Installing Tanuki (deprecated)

<Callout type="warning" emoji="⚠️">
    Note that this version requires a whole disk and is not compatible with dual-boot systems. It is best tried in a virtual machine.
</Callout>

<Steps>

### Boot the Live ISO
### Open `Filer` and navigate to `/Applications/Utilities`
### Double-click on `Install AiryxOS.app`

</Steps>

The installer will then walk you through the installation process.
